# ATLAS-HEMATOL-GICO
O Atlas Hematológico é uma plataforma interativa para estudantes de Análises Clínicas e Biomedicina. Reúne morfologia sanguínea com cards ilustrativos, vídeos, quiz com 30 questões e busca inteligente. Aborda hematopoiese, células maduras e alterações morfológicas, com feedback imediato e interface responsiva.
